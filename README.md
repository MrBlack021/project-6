# Huntifyy - AI & Startup Launchpad

This is a single-page application built with React.

## Getting Started

To run this project locally, you need to have Node.js and npm installed.

1.  **Install Dependencies:**
    ```bash
    npm install
    ```
2.  **Run the Development Server:**
    ```bash
    npm run dev
    ```
This will start the development server, and you can view the application in your browser.

## Building for Production

To create a production build of the application, run:
```bash
npm run build
```
This will generate optimized static assets in the `dist` folder.
